# block

## usage

```
Block IP by iptables

Usage: block [option]

Options:
  -a string
    	Specify insert/append/delete/clear/list rule (default "insert")
  -f string
    	Specify log file
  -ip string
    	Specify IP or CIDRs
  -limit int
    	Limit requests per minute (default 100)
  -o string
    	Specify Block in/out bound traffic or log (default "in")
```
